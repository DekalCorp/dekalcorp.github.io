What's under the hood?

Code Based Design Interface

Dekal let's you describe your kinematic structure through code. This let's you rapidely generate your design and variations of it. Use your favorite AI Copilot to accelerate the process.

Compiler & Automatic Differentiation

Similar to machine learning frameworks like PyTorch or JAX, the Dekal compiler generates a differentiable graph of your kinematic description. This is both used to power the kinematic and load solvers as well as to enable gradient descent design optimization in the future.

Kinematics Constraint System

Dekal includes a powerful constraint system which let's you design any kinematic structure you can think of, such as pistons, ropes, pulleys, capstan, gears and linkage systems.

Load Solver

Dekal uses lagrangian mechanics to resolves loads within your kinematic structures. Define a set of static loads on structure and dekal will resolves all the reaction forces and torques, which can be plotted for review.

Why build Dekal?

Dekal is built by a mechanical engineer who became frustrated with the status-quo in engineering design software. While working in the drone and automotive industry, existing software became the key limiting factor towards faster design iteration speeds.

Dekal uses the latest technology in compilers, machine learning and 3D rendering to break down those barriers for fellow engineers, working on the next exciting products.
